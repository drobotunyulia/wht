# Dataset

В экспериментах используется UC Merced Land Use Dataset.

Датасет содержит 21 класс изображений аэрофотосъёмки, по 100 изображений в каждом классе. Размер каждого изображения — 256×256 пикселей.

Оригинальный источник:

Yang, Y., Newsam, S. Bag-Of-Visual-Words and Spatial Extensions for Land-Use Classification. ACM SIGSPATIAL International Conference on Advances in Geographic Information Systems, 2010.

Датасет можно скачать с официальной страницы:
https://vision.ucmerced.edu/datasets/land-use-dataset

После скачивания изображения нужно поместить в папку `data/`.
