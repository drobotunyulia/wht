# JPEG-like Image Compression Based on Walsh-Hadamard Transform

Репозиторий содержит программную реализацию и экспериментальные скрипты для выпускной квалификационной работы магистра на тему:

**«Разработка метода малозатратного сжатия изображений аэрофотосъёмки для бортовых систем БПЛА на основе преобразования Уолша–Адамара»**

## Описание проекта

Цель работы — исследовать возможность применения преобразования Уолша–Адамара в JPEG-подобной схеме сжатия изображений аэрофотосъёмки.

В проекте реализованы:

- базовый JPEG-подобный алгоритм с использованием ДКП;
- JPEG-подобный алгоритм с заменой ДКП на преобразование Уолша–Адамара;
- поддержка матриц Адамара порядков 8×8, 16×16, 24×24 и 32×32;
- сравнение различных классов матриц Адамара порядка 16;
- исследование матрицы Пэли порядка 24;
- исследование матриц порядка 32 с Sylvester- и sequence-упорядочением;
- экспериментальная оценка коэффициента сжатия, PSNR, времени кодирования и декодирования.

## Структура проекта

```text

```

## Установка

Клонирование репозитория:


```text
git clone https://github.com/USERNAME/uav-image-compression-wht.git
```
```text
cd wht
```

Создание виртуального окружения:
```text
python3 -m venv .venv
```

```text
source .venv/bin/activate
```

Установка зависимостей:
```text
pip install -r requirements.txt
```
## Используемый набор данных

В работе используется UC Merced Land Use Dataset.

После загрузки изображения необходимо разместить в папке:
```text
data/
```
